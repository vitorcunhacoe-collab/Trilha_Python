# Trilha_Python
for_code
O programa funciona baseado em 2 grandes blocos, o primeiro em que o usuário fornece as informações de uma possível viagem,como: Orçamento, destino, diária de hospedagem, custo da passagem aérea e tempo da viagem. Após o fornecimento desses dados de acordo com o formato de dados requisitado faz-se necessário validar se os valores fornecidos não são negativos, visto que valores negativos seriam claros erros. Dessa forma a execução do código é encerrada quando algum valor é negativo.
No segundo bloco, são realizadas varias contas de modo a fornecer no final ao usuário se o orçamento é possível ou não, o que condicionado juntamente ao número de dias da viagem ser maior que 0 determina se a viagem é viável ou inviável.
Para utilizar o código basta executá-lo e fornecer as informações requisitadas no terminal.
